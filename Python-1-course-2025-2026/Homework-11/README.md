# Домашнее задание 11
## Тема 12. Python для аналитиков ИБ: поиск негативных событий

### Проанализировать два типа логов изпредставленного набора данных, найти подозрительные события и визуализировать 10 наиболее распространённых подозрительных событий.

Описание
В файле botsv1.json содержатся логи формата JSON.
Из WinEventLog анализируются подозрительные EventID: 4625, 4673, 4674, 4688, 4697, 7045.
В ходе анализа найдено одно подозрительное событие: EventID 4688 (9 раз).
Построена визуализация количества найденных событий.
Также выполнен базовый анализ DNS‑логов (длина домена, количество поддоменов).

Запуск
Откройте проект в Jupyter Notebook.
Установите зависимости:

pip install pandas seaborn matplotlib

Запустите ноутбук:

jupyter notebook analysis.ipynb
