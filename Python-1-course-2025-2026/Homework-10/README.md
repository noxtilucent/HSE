# Домашнее задание 10 Программирование на Python (1 курс, 2025/2026) 
## Тема 11. Python для аналитиков ИБ: эксплойты

### Найдите описание CVE-уязвимости (common vulnerabilities and exposures)
и напишиеминимальныйPoC-скрипт (proof of concept), который эмулирует
использование уязвимости (можно без её активации).
В скрипте достаточно имитировать взаимодействие, например,
формирование запроса к условной уязвимой точке, и выводить сообщение
о потенциальной атаке.

### Описание
CVE‑2021‑41773 — уязвимость обхода пути (Path Traversal) в Apache HTTP Server 2.4.49. Она позволяет злоумышленнику получить доступ к файлам вне корневой директории веб‑сервера, если включён Require all granted для определённых путей.

# Запуск 
Откройте проект в VS Code или PyCharm.

Убедитесь, что установлен модуль requests:
bash
pip install requests

Запустите PoC‑скрипт:
bash
python poc.py

Скрипт отправит тестовый запрос к условному серверу и выведет сообщение о потенциальной уязвимости.
